The Interfacer: a bot and API allowing developers to access information about Discord servers and users while ensuring privacy.
Invite Interfacer.

How can I use the Interfacer for servers?

  1. Invite the interfacer to your server using the link above OR Check that the Interfacer bot is in the server you'd like to access.
  2. Make a request to the API endpoint /guilds/<guild_id>.
  3. Get the response!

How can I use the Interfacer for individual users?

  1. Invite the interfacer to your server using the link above OR Check that the Interfacer bot is in the server you'd like to access.
  2. Opt in for usage with the !optin command in Discord. You can always opt out with !optout.
  3. Make a request to the API endpoint /guilds/<guild_id>/users/<user_id>.
  4. Get the response!
